# Movie-MVVM
Hello everyone!
I want to share with you my Movie-MVVM Trial App.
<br>
I tried to develop this application by adhering to the following items.
<br><br>

- UIKit
- Tableview
- CustomTableViewCell

- BaseNavigationController
- Alamofire
- Kingfisher
- CustomCollectionCell
- Search Controller & Search Bar
